Import Stock from Spreadsheet

Prepare an Excel/Google Sheet with these column headers in the first row, then Save As → CSV and upload it here:

barcode, name, category, karat, weight_grams, cost_price, location, quantity

  • barcode — the code on the tag (leave blank to auto-generate a JW code)
  • category — ring / necklace / bracelet / earring / chain / pendant / other
  • karat — 18, 21, or 24
  • weight_grams — e.g. 5.50
  • cost_price — what the piece cost you (EGP)
  • location — safe or showcase
  • quantity — usually 1
